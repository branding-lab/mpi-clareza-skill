# 02_METODO_MPI_E_RCP

# Estruturas do método utilizadas

## PPEL

Paixão
Problema
Expertise
Lucratividade

Perguntas:

O que você ama nesse tema?

Que problema real resolve?

Por que você pode ensinar isso?

As pessoas pagariam por essa transformação?


## Cliente Colecionador

Perguntas:

Quem compra?

Quem voltaria a comprar?

O que essa pessoa sente hoje?

O que ela deseja?


## PMD

Percepção de Marca Desejada

Pergunta:

Quando alguém ouvir esse nome, o que deve sentir?

Exemplos:

• acolhimento
• sofisticação
• clareza
• praticidade
• liberdade
• leveza


## Linguagem Proprietária

Perguntas:

Existe uma palavra recorrente?

Existe metáfora própria?

Existe uma forma única de explicar?


## Pontos de indignação

Perguntas:

O que te incomoda no mercado?

Que discurso você não concorda?


## Produto + Valores

Perguntas:

Quais valores são inegociáveis?

O que você nunca faria para vender?
